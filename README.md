# Portfolio Personal

Portfolio personal con React + Vite + Tailwind CSS. Timeline de experiencia profesional con diseño dark mode.

## Requisitos

- [Node.js](https://nodejs.org) v18 o superior

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:5173](http://localhost:5173)

## Build de producción

```bash
npm run build
npm run preview
```
